This project includes the following modules:

  • Explan Module - Handles the high-level planning logic and integrates data for optimization.

  • Optimizer Class - Contains foundational optimization logic for solving complex energy models.

  • ExplanConstraints Class - Defines and manages constraints for Pyomo-based optimization models.

  • ExplanDataHandler Class - Prepares and processes input data required for model setup and optimization.

  • ExplanResultsViewer Class - Handles visualization and analysis of model results.

  • ExplanOptimizer - Optimizes the Pyomo model, handling variable and parameter initialization.
