Explan Module¶

The Explan module is a core component of the QuESt Planning framework, providing tools for energy system planning and optimization. This module focuses on managing data, defining optimization constraints, and handling results visualization.

It serves as the backbone for integrating data handling and Pyomo-based optimization processes.

Key Features: - Interfaces with data handlers for loading and processing system input data. - Defines mathematical constraints and parameters for energy planning problems. - Configures and solves optimization models using Pyomo. - Provides utilities for viewing and analyzing optimization results.

Submodules¶

The Explan module contains the following submodules:

  • ExplanConstraints Class - Defines mathematical constraints for optimization problems.

  • ExplanDataHandler Class - Handles input data processing and parameter preparation.

  • ExplanResultsViewer Class - Provides tools for visualizing and analyzing optimization results.

  • ExplanOptimizer - Configures and solves Pyomo optimization models.
